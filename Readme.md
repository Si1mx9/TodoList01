# TodoList

A modern task management app built with React, TypeScript, and Vite.

## Features

- Create and manage todos with titles, descriptions, due dates, and priorities
- Organize tasks into projects with custom colors
- Filter by today, uncompleted, or view all
- Sort by date, priority, creation, or title
- Search through todos
- Checklist items within todos
- Archive/unarchive projects
- Responsive design with mobile sidebar
- Data persisted to localStorage

## Tech Stack

- **React 19** with TypeScript
- **Vite** for bundling
- **Tailwind CSS** + **shadcn/ui** components
- **date-fns** for date formatting
- **lucide-react** for icons
- **framer-motion** for animations
- **react-hook-form** + **zod** for forms and validation

## Getting Started

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
npm run preview
```
